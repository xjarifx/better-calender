# Tech Stack

- **Framework**: Next.js (full-stack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS with OKLCH color system
- **Database**: PostgreSQL with Prisma ORM
- **Auth**: Username/password with bcrypt + JWT or sessions
- **AI**: OpenRouter API
- **Color Palette**: Google Calendar-inspired dark theme (see `docs/COLOR_PALETTE.md`)
